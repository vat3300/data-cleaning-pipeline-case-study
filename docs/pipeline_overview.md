# Data Cleaning Pipeline Overview

## Input
Raw operational data containing missing values, duplicates, inconsistent formats, and invalid records.

## Processing Steps
1. Load raw CSV data
2. Standardize column naming
3. Handle missing customer, region, and amount values
4. Normalize text fields
5. Convert date fields into standard datetime format
6. Remove duplicate records
7. Filter invalid numeric values
8. Apply basic validation checks

## Output
A clean, analytics-ready dataset suitable for reporting and dashboard creation.
